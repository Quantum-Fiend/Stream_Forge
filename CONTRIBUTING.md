# Contributing to StreamForge

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make changes following code style guidelines
4. Add tests: `./gradlew test`
5. Commit: `git commit -m "feat: add feature"`
6. Push and create PR

## Code Style

- **Scala**: Functional, immutable, pure functions
- **Java**: Standard conventions, JavaDoc for public APIs
- **Kotlin**: Kotlin conventions, coroutines for async
- **Dart**: Dart style guide, meaningful widget names

## Questions?

Open an issue or reach out to maintainers.
